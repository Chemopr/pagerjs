Cached Lazy Loaded Content

Lazy loaded content is possible to cache. Data binding is also working into this file. The view models name () is data bound in this text. Add sourceCache: true 

<div data-bind="page: {
      id: 'cached_lazy',
      sourceCache: true,
      sourceOnShow: 'cached_lazy.html .content'}">
    ...
</div>
